\# BSC Web3 - Next.js Smart Contract Interaction Platform



A modern, production-ready Web3 decentralized application built with Next.js, ethers.js, and wagmi for interacting with smart contracts on Binance Smart Chain (BSC).



\## Features



✨ \*\*Wallet Connection\*\*

\- MetaMask and multiple wallet support via RainbowKit

\- Automatic network detection and BSC validation

\- Real-time balance display



🔗 \*\*Smart Contract Interaction\*\*

\- Read contract data with view/pure functions

\- Execute state-modifying transactions

\- Gas estimation and monitoring

\- Error handling and user feedback



🎨 \*\*Modern UI\*\*

\- Built with Tailwind CSS

\- Responsive design for all devices

\- Real-time wallet information display



⚙️ \*\*Developer-Friendly\*\*

\- TypeScript support

\- Hooks-based architecture

\- Reusable utility functions



\## Getting Started



1\. Install dependencies: `npm install`

2\. Start dev server: `npm run dev`

3\. Open http://localhost:3000 and connect wallet



\## Key Files



\- `src/config/web3Config.ts` - Web3 and contract configuration

\- `src/providers/Web3Providers.tsx` - Wagmi + RainbowKit setup

\- `src/components/` - UI components

\- `src/hooks/` - Custom React hooks

\- `src/lib/` - Contract interaction utilities



```bash

npm run dev

\# or

yarn dev

\# or

pnpm dev

\# or

bun dev

```



Open \[http://localhost:3000](http://localhost:3000) with your browser to see the result.
